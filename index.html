Berikut tips untuk lebih produktif dalam programming

Pintasan Dasar Editor:

  • Ctrl + P: Buka dengan cepat, memungkinkan Anda untuk dengan cepat menjelajahi file di proyek Anda.
  • Ctrl + Shift + P: Command Palette, di mana Anda dapat mengakses berbagai perintah dan pengaturan.
  • Ctrl + B: Alihkan visibilitas sidebar.
  • Ctrl + `: Alihkan terminal terintegrasi.

Pengelolaan File dan Workspace:

  • Ctrl + N: File baru.
  • Ctrl + O: Buka file.
  • Ctrl + S: Simpan file.
  • Ctrl + Shift + S: Simpan file sebagai.
  • Ctrl + W: Tutup file.
  • Ctrl + Shift + T: Buka kembali file yang baru saja ditutup.
  • Ctrl + Shift + N: Jendela baru.
  • Ctrl + Shift + W: Tutup jendela.

Pengeditan dan Navigasi:

  • Ctrl + D: Pilih kemunculan berikutnya dari seleksi saat ini.
  • Ctrl + F: Cari.
  • Ctrl + H: Ganti.
  • Ctrl + G: Buka ke baris tertentu.
  • Ctrl + P + @: Buka ke simbol.
  • Ctrl + ] / [: Indentasi/Deindentasi baris atau seleksi.
  • Alt + ↑ / ↓: Pindahkan baris ke atas atau ke bawah.
  • Ctrl + /: Alihkan komentar pada baris.
  • Ctrl + Shift + /: Alihkan komentar blok.

Pemformatan Kode:

  • Shift + Alt + F: Pemformatan kode.
  • Ctrl + K + F: Pemformatan blok kode yang dipilih.

Terminal Bawaan:

  • Ctrl + `: Tampilkan atau sembunyikan terminal terintegrasi.
  • Ctrl + Shift + `: Buat instansi terminal baru.

Debugging:

  • F5: Mulai atau lanjutkan debugging.
  • F9: Alihkan breakpoint.
  • F10: Langkah di atas.
  • F11: Langkah masuk.
  • Shift + F11: Langkah keluar.

Course 1

jenis-jenis semantic tag Html

<header>

MTag ini digunakan untuk mewakili konten pengantar atau kelompok tautan navigasi di bagian atas sebuah halaman atau bagian halaman.

<header> <h1>Selamat datang di situs saya</h1> <nav> <a href="/">Beranda</a> <a href="/tentang">Tentang</a> <a href="/kontak">Kontak</a> </nav> </header>

<nav>

Tag ini digunakan untuk mendefinisikan sebuah Elemen atau Pembungkus konten yang berisi tautan navigasi.

<nav> <a href="/">Beranda</a> <a href="/tentang">Tentang</a> <a href="/kontak">Kontak</a> </nav>

<main>

Menandai konten utama sebuah halaman dan harus unik dalam satu halaman.

<main> <h3>Artikel Terbaru</h3> <article> <h3>Judul Artikel</h3> <p>Isi artikel...</p> </article> </main>

<article>

Tag ini merepresentasikan komposisi mandiri yang dapat didistribusikan atau digunakan kembali. Biasanya digunakan untuk mengelompokkan konten artikel, blog post, atau posting media lainnya.

<article> <h3>Judul Artikel</h3> <p>Isi artikel...</p> </article>

<section>

Tag ini menandai sebagian mandiri dari sebuah dokumen atau aplikasi, Biasanya digunakan untuk mengelompokkan konten terkait dalam satu bagian.

<section> <h3>Produk</h3> <article> <h3>Produk 1</h3> <p>Deskripsi produk...</p> </article> <article> <h3>Produk 2</h3> <p>Deskripsi produk...</p> </article> </section>

<aside>

Merepresentasikan konten yang berkaitan dengan konten utama secara sekunder.

<aside> <h3>Artikel Populer</h3> <ul> <li>Artikel 1</li> <li>Artikel 2</li> </ul> </aside>

<footer>

Mendefinisikan footer atau bagian bawah dari sebuah dokumen atau sektion.

<footer> <p>Hak Cipta © 2023 Nama Situs</p> </footer>

<h1> sampai <h6>

Menandai judul dengan tingkat kepentingan yang berbeda.

<h1>Judul Utama</h1> <h3>Judul Seksi</h3>

<p>

Menandai paragraf teks.

<p>Ini adalah paragraf teks.</p>

<ul>

Membuat daftar tidak terurut.

<ul> <li>Poin 1</li> <li>Poin 2</li> </ul>

<ol>

Membuat daftar terurut.

<ol> <li>Poin 1</li> <li>Poin 2</li> </ol>

<li>

Menandai item dalam sebuah daftar.

<ul> <li>Poin 1</li> <li>Poin 2</li> </ul>

<a>

Membuat tautan atau hyperlink.

<a href="https://www.contoh.com">Tautan</a>

<img>

Menyisipkan gambar ke dalam halaman web.

<img src="gambar.jpg" alt="Deskripsi Gambar">

<figure>

Merepresentasikan konten mandiri, seperti gambar, ilustrasi, atau potongan kode.

<figure> <img src="gambar.jpg" alt="Deskripsi Gambar"> <figcaption>Gambar ilustrasi</figcaption> </figure>

<figcaption>

Menyediakan keterangan atau deskripsi untuk konten dalam elemen <figure>.

<figure> <img src="gambar.jpg" alt="Deskripsi Gambar"> <figcaption>Gambar ilustrasi</figcaption> </figure>

<table>

Membuat tabel untuk data tabular.

<table> <tr> <th>Nama</th> <th>Usia</th> </tr> <tr> <td>John</td> <td>25</td> </tr> </table>

<thead>, <tbody>, <tfoot>

Mendefinisikan bagian kepala, isi, dan kaki tabel.

<table> <thead> <tr> <th>Nama</th> <th>Usia</th> </tr> </thead> <tbody> <tr> <td>John</td> <td>25</td> </tr> </tbody> <tfoot> <tr> <td colspan="2">Total</td> </tr> </tfoot> </table>

<tr>

Menandai baris dalam sebuah tabel.

<tr> <td>Data 1</td> <td>Data 2</td> </tr>

<th>

Menandai sel kepala (header) dalam sebuah tabel.

<tr> <th>Nama</th> <th>Usia</th> </tr>

<td>

Menandai sel data dalam sebuah tabel.

<tr> <td>Data 1</td> <td>Data 2</td> </tr>

<form>

Membuat formulir dengan elemen-elemen interaktif untuk mengirim data dari pengguna.

<form action="/proses" method="post"> <label for="nama">Nama:</label> <input type="text" id="nama" name="nama" required> <br> <label for="email">Email:</label> <input type="email" id="email" name="email" required> <br> <input type="submit" value="Kirim"> </form>

<input>

Membuat elemen input dalam sebuah formulir.

<input type="text" id="nama" name="nama" required> <input type="checkbox" id="setuju" name="setuju">

<label>

Menandai label yang terkait dengan elemen <input>.

<label for="nama">Nama:</label> <input type="text" id="nama" name="nama" required>

<button>

Membuat tombol yang dapat diklik.

<button type="button">Klik Saya</button>

<textarea>

Membuat input teks dengan beberapa baris.

<textarea id="pesan" name="pesan" rows="4" cols="40"></textarea>

<select>

Membuat kotak pilihan atau dropdown.

<select id="negara" name="negara"> <option value="id">Indonesia</option> <option value="us">Amerika Serikat</option> </select>

<option>

Menandai pilihan dalam elemen <select>.

<select id="negara" name="negara"> <option value="id">Indonesia</option> <option value="us">Amerika Serikat</option> </select>

<div>

Wadah generik untuk mengelompokkan dan mengatur elemen-elemen dalam halaman.

<div class="container"> <h1>Judul</h1> <p>Konten...</p> </div>
